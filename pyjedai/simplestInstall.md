# Simplest Way to run pyjedai
### Requirements
- gcc
- g++

### Installation Instruction

Install
```
pip install pyjedai
```

Download Abt-Buy Dataset

```
wget https://dbs.uni-leipzig.de/files/datasets/Abt-Buy.zip
unzip Abt-Buy.zip
rm Abt-Buy.zip
```

Run Clean Clean Pipeline Test
```
python cleanclean.py
```
